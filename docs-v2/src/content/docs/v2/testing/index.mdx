The default v2 test loop is Rust-first. Instead of starting a local validator and driving it from TypeScript, the scaffolded test loads the compiled SBF program into LiteSVM inside the Rust test process.

`anchor-v2-testing` wraps LiteSVM, `$ <blue>anchor</blue> test <dim>--profile</dim>` renders flamegraphs, `$ <blue>anchor</blue> debugger` opens an instruction stepper, and `$ <blue>anchor</blue> coverage` emits LCOV from the same register traces. One test harness covers normal tests, performance work, debugging, and coverage.
