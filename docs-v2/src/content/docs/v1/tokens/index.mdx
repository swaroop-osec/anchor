## What are token programs?

Two token programs developed by [Anza](https://www.anza.xyz/) (previously Solana Labs) issue and manage fungible and non-fungible tokens on Solana. The original [Token Program](https://github.com/solana-program/token) ships immutable and is the most widely used; it covers the basics like minting and transferring. The newer [Token Extension Program (Token 2022)](https://github.com/solana-program/token-2022) is a superset that adds optional behaviors through "extensions" and is the recommended choice for new tokens.

## Invoking token programs in an Anchor program

The [`anchor-spl`](https://github.com/otter-sec/anchor/tree/62865c636aecc6974fc9cfebfc6cf08ca4f0bb72/spl) crate simplifies interacting with Solana's Token Programs from an Anchor program. It ships instructions and account types for both the original Token Program and the newer Token Extension Program (Token 2022).

Add the `anchor-spl` crate as a dependency and add `"anchor-spl/idl-build"` to the `idl-build` feature list in `Cargo.toml{:file}`. See the [quickstart page](/docs/v1/get-started/local-development/) for a walkthrough of creating an Anchor program locally:

```console
$ <blue>cargo</blue> add anchor-spl
```

```toml title="Cargo.toml"
[features]
idl-build = [
    "anchor-lang/idl-build",
    "anchor-spl/idl-build",
]

[dependencies]
anchor-lang = "1.0.1"
anchor-spl = "1.0.1"
```

The most commonly used modules in `anchor-spl`:

| Module | Description |
| --- | --- |
| [`token`](https://github.com/otter-sec/anchor/blob/62865c636aecc6974fc9cfebfc6cf08ca4f0bb72/spl/src/token.rs) | Token Program (legacy) instructions and account types |
| [`token_2022`](https://github.com/otter-sec/anchor/blob/62865c636aecc6974fc9cfebfc6cf08ca4f0bb72/spl/src/token_2022.rs) | Token 2022 base instructions (matching the Token Program functionality) |
| [`token_2022_extensions`](https://github.com/otter-sec/anchor/tree/62865c636aecc6974fc9cfebfc6cf08ca4f0bb72/spl/src/token_2022_extensions) | Token 2022 extensions instructions |
| [`token_interface`](https://github.com/otter-sec/anchor/blob/62865c636aecc6974fc9cfebfc6cf08ca4f0bb72/spl/src/token_interface.rs) | Account types that work with both the Token Program and Token 2022 Program |
| [`associated_token`](https://github.com/otter-sec/anchor/blob/62865c636aecc6974fc9cfebfc6cf08ca4f0bb72/spl/src/associated_token.rs) | Associated token account instruction |
